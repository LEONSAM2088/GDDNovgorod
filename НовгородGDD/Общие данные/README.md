# Вече: Новгородская Сага — база знаний

Obsidian-хранилище по разработке игры. Ниже — карта папок.

## Структура

```
РазработкаИгр/
├── README.md                 ← вы здесь
│
├── GDD/                      Геймдизайн игры
│   ├── Первый GDD.md         Главный дизайн-документ
│   ├── Сеттинг/              Исторический материал (XIV–XV в.)
│   │   └── Сословия ранги и функции/  Сословия, ранги, должности (для GDD)
│   ├── Механики/             Детальные системы (экономика, NPC, слухи…)
│   ├── Контент/              Игровой контент (рецепты, предметы…)
│   └── Идеи/                 Заметки-идеи по геймплею
│
├── Принципы дизайна/         Общие принципы геймдизайна (не привязаны к одной механике)
│
├── анализ/                   Исследование рынка и конкурентов
├── Проект/                   Планирование: аудитория, стэк, рынок, ГДД, DLC
│
├── Разработка/               Техническая и производственная сторона
│   ├── Техника/              Код, оптимизация, сохранения, локализация, a11y
│   └── Продакшен/            Музыка, полировка, титры
│
├── Финансирование/           Инвестиции, издатели, краудфандинг, гранты
│
├── Настрой/                  Мотивация, цели, стратегия студии (SMART и др.)
│
├── Ресурсы/                  Внешние материалы
│   └── Вложения/             PDF и прочие файлы
│
└── Журнал/                   Дневниковые заметки по датам
```

## Правила именования

- **GDD/** — всё, что напрямую описывает игру «Вече».
- **Принципы дизайна/** — универсальные мысли о геймдизайне (подходят для любого проекта).
- **Проект/** — бизнес и планирование разработки.
- **Разработка/** — как строить и выпускать игру технически.
- **Настрой/** — личная дисциплина и цели как разработчика/студии.

Ссылки `[[как в Obsidian]]` работают по имени файла — путь в ссылке указывать не обязательно.
