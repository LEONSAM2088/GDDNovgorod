## Ранги — не только сословия

В Новгороде статус складывается из **сословия**, **имущества**, **должности** и **принадлежности к корпорации** (ста, слободе, концу). Один человек может быть одновременно купцом, землевладельцем и участником суда.

## Городские звания внутри сословия

### Купцы: «старейшие» и «молодшие»

- **Старейшие** — головы корпораций, судьи «братчины», хранители мер
- **Молодшие** — рядовые члены ста, претенденты на старшинство
- Переход — через богатство, стаж, брак, покровительство боярина

### Ремесленники: «старейшие» и «чёрные»

- **Старейшие** — уважаемые мастера слобод
- **Чёрные** (в городском контексте) — младший слой, не «передние» бояре
- Термин **не этнический** — чисто социальный

## Должностные звания (не наследуемые)

| Звание | Отношение к рангу |
|--------|-------------------|
| **Посадник** | Из бояр; «старый посадник» остаётся в совете |
| **Тысяцкий** | Из знатных боярских семей |
| **Сотский** | Выборный от сотни; полиция и суд на улице |
| **Кончанский староста** | Выборный от конца; коллегия знатных обывателей |
| **Новгородский муж** | Управляет волостью в пятине |
| **Пристав, бирить, изветник** | Исполнители посадника и тысяцкого |

## Церковные ранги с политическим весом

| Ранг | Функция |
|------|---------|
| Архиепископ (владыка) | Глава церкви и совета |
| Архимандрит Юрьева монастыря | Заменяет владыку на совете |
| Настоятели крупных монастырей | Кандидаты в владыки, землевладельцы |
| Белое и чёрное духовенство | Суд владыки, десятина, грамотность |

## Формула представительства на вече

**Бояре → житьи люди → купцы → чёрные люди → все пять концов**

Игрок должен понимать: подъём по лестнице = доступ к **голосу на вече** и к **должностям**.

## Для игры

| Механика | Исторический прототип |
|----------|----------------------|
| Репутация в слободе | Старейший ремесленник |
| Членство в гильдии | Ста / Иванское сто |
| Титул vs власть | Боярин без должности слабее посадника-«чужого» рода |
| Два пути вверх | Богатство (купец) или земля (своеземец → житой) |
| Потолок без связей | Чёрный человек не станет посадником без смены статуса |
