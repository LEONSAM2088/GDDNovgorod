## Ключевые тезисы статьи

### Исторический контекст

- 1136 г. — изгнание князя, боярское господство; республика до 1478 г.
- Торговый центр: Ганза, Балтика, Восток; колониальные владения на севере
- Княжеского **домена** в Новгородской земле не сложилось

### Социальные слои (кратко)

| Слой | Суть |
|------|------|
| Бояре | Крупные феодалы, торговля, ростовщичество |
| Владыка и церковь | Земля, десятина, казна, политика |
| Житьи люди | Средние феодалы-торговцы |
| Своеземцы | Мелкие землевладельцы |
| Купцы | Сотни и ста; Иванское сто |
| Ремесленники | Слободы и сотни |
| Смерды | Государственные земли |
| Половники | Изорники, огородники, кочетники |
| Закладники, холопы | Низ зависимости |

### Государственный строй (формула автора)

- Форма: **боярская аристократическая республика**
- Устройство: **федерация** (концы + пятины)
- Режим: элементы **демократии** при реальной власти бояр

### Органы

- **Вече** — верховная власть юридически; единогласие; кворум
- **Оспода** — реальные решения
- **Посадник, тысяцкий, князь, владыка** — см. файлы 03–04

### Администрация

- 5 концов → 10 сотен → улицы
- 5 пятин → волости → погосты
