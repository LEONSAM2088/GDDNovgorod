## Пятина

**Пятина** — крупнейшая административная единица **Новгородской земли** (не «доля» в бытовом смысле).

|                     |                                                              |
| ------------------- | ------------------------------------------------------------ |
| **Что это**         | Радиальная полоса территории — «луч» от Великого Новгорода   |
| **Сколько**         | 5: Деревская, Обонечская, Бежецкая, Подинская, Русская       |
| **Внутри**          | [[#Полупятина\|полупятины]], [[#Погост\|погосты]], волости   |
| **Зачем**           | Перепись, [[#Подать\|подати]], суд, сбор ополчения           |
| **Связь с городом** | Каждая пятина связана с одним [[#Конец (городской)\|концом]] |

**Аналогия для игры:** регион на карте — оттуда идут сырьё, подати и ополчение.

**Подробнее:** [[04 — Территория и администрация#Пятины (земля)]]

**Упоминается в:** [[00 — Обзор]], [[03 — Сословия и общество]], [[05 — Экономика и торговля]], [[09 — Внешняя политика]]

## Конец (городской)

**Конец** — район Великого Новгорода; административная и общинная единица.

| Конец | [[#Пятина\|Пятина]] |
|-------|---------------------|
| Плотницкий | Деревская |
| Славенский | Обонечская |
| Загородский | Бежецкая |
| Неревский | Подинская |
| Гончарный | Русская |

Делится на [[#Сотня|сотни]] и улицы.

**Упоминается в:** [[04 — Территория и администрация]], [[01 — Политическая система]], [[02 — Вече и должности]]
