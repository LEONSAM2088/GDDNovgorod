# Контекст проекта для анализа рынка

> Снимок концепции на момент начала анализа. Источник: [[Первый GDD]] v1.1.

## Идентификация

| Параметр | Значение |
|----------|----------|
| Рабочее название | **Вече: Новгородская Сага** |
| Жанр | Средневековый RPG-симулятор + стратегия + династическое управление |
| Сеттинг | Великий Новгород, XIV век |
| Масштаб | Инди (предположительно; уточнить) |
| Целевые платформы | *TBD — см. [[05 — Платформы и дистрибуция]]* |

## Ключевые особенности (USP для позиционирования)

- Бесконечная династическая игра в живом simulation-based мире
- Социальный лифт: от смерда до посадника / князя / архиепископа
- NPC на равных с игроком: конкурируют, стареют, интригуют
- Гибкое происхождение вместо жёстких классов
- Экономика: цепочки поставок, качество товаров, риски для неграмотных
- Грамотность как социальный лифт (аналог Kingdom Come)
- Бесшовный мир без загрузок при входе в здания

## Целевая аудитория (гипотеза)

| Сегмент | Почему может заинтересоваться |
|---------|-------------------------------|
| Фанаты grand strategy / 4X | Династии, политика, долгая кампания |
| Фанаты life sim / sandbox | Живой мир, социальные связи |
| Любители исторических игр | Уникальный сеттинг — Новгород, не «очередная Европа» |
| Игроки CK / The Guild / Norland | Прямые аналоги по механикам |

**Гипотеза по демографии:** преимущественно мужская аудитория 25–45 лет, интерес к истории и сложным симуляторам. *Проверить в [[05 — Платформы и дистрибуция]].*

## Прямые аналоги (из GDD)

1. **Crusader Kings** — династии, интриги, политика
2. **The Guild** — социально-экономический симулятор средневековья
3. **Norland** — династия, классовая борьба, живой мир
4. **Rimworld** — симуляция колонии, emergent gameplay
5. **Victoria** — экономика, политика, социальные слои
