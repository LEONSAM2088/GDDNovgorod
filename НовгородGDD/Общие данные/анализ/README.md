## Быстрый старт

1. Прочитать [[00 — Контекст проекта]] — зафиксировать, что именно анализируем.
2. Заполнить [[05 — Платформы и дистрибуция]] — выбрать целевые площадки (Steam, GOG и т.д.).
3. Создать карточки для каждого аналога из GDD в папке `Конкуренты/`.
4. Свести выводы в [[06 — Прогноз и выводы]].

## Полезные источники данных

- SteamDB, Steam Charts — продажи, одновременные игроки, теги
- Gamalytic, VG Insights — оценки выручки Steam
- Steam Spy (архив) / SteamDB owner estimates
- HowLongToBeat — длительность прохождения
- IGN, PC Gamer, Rock Paper Shotgun — обзоры жанра
- Отчёты Newzoo, Statista — макротренды (платно/сводки)
- GDC talks, postmortems инди-студий
