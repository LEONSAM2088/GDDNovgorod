#Данные
# Зачем

Доверие нужно, чтобы игрок мог получать то, чего нельзя купить деньгами сразу: отсрочку, кредит, союз, голос, веру в слово и в слух.
# Где живёт

Объект: [[Игровой персонаж (Chapter)]]
# Схема

```json
{
	"trustLevels": [
		{"chapterId": int, "trustLevel": int}
	] // Уровень доверия от 0 до 100 к этому пероснажу
}
```

# Словари

**trustLevel** - Уровень доверия
```sql 
trustLevel >= 0 and trustLevel <= 100, DEFAULT 0, TYPE INT
```

**chapterId** - id персонажа
