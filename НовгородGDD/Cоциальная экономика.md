# Социалка — это не диалоги

Игрок не "общается" с NPC.

Игрок **строит сеть отношений**.

Не как в The Sims:

> Пошутить → +5 дружбы

А как в жизни:

> "Этот человек может быть мне полезен, а я могу быть полезен ему."

---

# Главный принцип

**Каждый NPC — самостоятельный человек со своими интересами.**

Он не существует ради игрока.

У него есть:

- семья;
- работа;
- друзья;
- враги;
- амбиции;
- страхи;
- обязанности;
- планы.

Он каждое решение принимает исходя из своих интересов.

---

# Социальные ресурсы

Как в экономике есть деньги и товары, так и в социалке есть свои ресурсы.

Например:

- доверие;
- репутация;
- информация;
- связи;
- влияние;
- обязательства;
- семейные союзы;
- секреты.

Все они могут накапливаться, теряться и использоваться.

---

# Главный вопрос игрока

Игрок постоянно думает:

> Кто мне может помочь?

> Чем я могу быть полезен этому человеку?

> Стоит ли ему доверять?

> Какие у него цели?

---

# Социальная экономика

Социалка работает почти как рынок.

Ты можешь:

   #Социальныерычаги
   Отдать:

- помощь;
- информацию;
- деньги;
- защиту;
- рекомендацию;
- поддержку.

Получить:

- работу;
- кредит;
- контракт;
- доступ;
- покровительство;
- брак;
- политическую поддержку.

То есть отношения становятся обменом ценностями, а не полоской "дружбы".

---

# Информация — отдельный ресурс

Игрок никогда не знает всего.

Он постепенно узнаёт:

- кто с кем дружит;
- кто кому должен;
- кто ищет работников;
- кто хочет женить сына;
- кто разоряется;
- кто собирается участвовать в политике.

Хорошая информация открывает возможности.

Плохая информация может привести к краху.

---

# Доверие ≠ выгода

Очень важно разделить.

Человек может:

- уважать тебя;
- не любить тебя;
- бояться тебя;
- сотрудничать с тобой;
- быть твоим родственником.
==У NPC эти статусы могут быть комбинированными, например: npc может уважать тебя и бояться или быть твои родственником и не любить тебя==

Это разные вещи.

Например:

Конкурент может тебя ненавидеть.

Но продолжать покупать твой товар.

Потому что ему это выгодно.

---

# Репутация семьи

Репутация принадлежит не только персонажу.

Она принадлежит роду.

Например:

Твой дед всегда выполнял обещания.

Через 50 лет:

Твоему сыну легче получить кредит.

Или наоборот.

Если семья известна мошенничеством:

Даже честному наследнику никто не доверяет.

---

# NPC помнят поступки

Не обязательно каждую мелочь.

Но ключевые события.

Например:

- помог открыть дело;
- предал;
- спас от банкротства;
- женился в их семье;
- отказал в помощи.

Это формирует историю отношений.

---

# Семьи важнее отдельных людей

Игрок взаимодействует не только с человеком.

Он взаимодействует с целым родом.

Например:

Женившись на дочери купца, ты приобретаешь:

- союзников;
- новых родственников;
- новые обязательства;
- новых врагов.

---

# Социалка влияет на экономику

Можно быть хорошим мастером.

Но без знакомств:

- не получить выгодный заказ;
- не найти поставщиков;
- не получить кредит.

Можно быть бедным.

Но иметь отличную репутацию.

И тогда люди сами предложат помощь.

---

# Социалка влияет на политику

Политика строится не на голосах.

А на отношениях.

Кто тебя поддержит?

Кому ты помог?

Кто тебе обязан?

Кто боится твоего влияния?

# Главная мысль игры

Игрок растёт **не потому, что набил уровень**, а потому, что **научился понимать людей**.

Он замечает #Трейты NPC:

| Трейт        | Противоположный трейт |
| ------------ | --------------------- |
| Амбициозен   | Малопритязательный    |
| Щедрый       | Жадный                |
| Смелый       | Трусливый             |
| Миролюбивый  | Гневный               |
| Справедливый | Лживый                |
| Наивный      | Хитрый                |
| Общительный  | Застеничивый          |
Каждый NPC имеет комбинацию из 3 вариантов трейтов, однако не может иметь в комбинации трейты противоречащие друг другу, т.е нельзя быть амбициозным и малопритязательным одновременно.

Пример комбинаций:
Амбициозен + Щедрый + Смелый  
Амбициозен + Щедрый + Трусливый  
Амбициозен + Жадный + Смелый  
Амбициозен + Жадный + Трусливый

Амбициозен + Миролюбивый + Справедливый  
Амбициозен + Миролюбивый + Лживый  
Амбициозен + Гневный + Справедливый  
Амбициозен + Гневный + Лживый

Амбициозен + Наивный + Общительный  
Амбициозен + Наивный + Застенчивый  
Амбициозен + Хитрый + Общительный  
Амбициозен + Хитрый + Застенчивый

Малопритязательный + Щедрый + Смелый  
Малопритязательный + Щедрый + Трусливый  
Малопритязательный + Жадный + Смелый  
Малопритязательный + Жадный + Трусливый

Малопритязательный + Миролюбивый + Справедливый  
Малопритязательный + Миролюбивый + Лживый  
Малопритязательный + Гневный + Справедливый  
Малопритязательный + Гневный + Лживый

import itertools
import json

pairs = [
    ["Амбициозен", "Малопритязательный"],
    ["Щедрый", "Жадный"],
    ["Смелый", "Трусливый"],
    ["Миролюбивый", "Гневный"],
    ["Справедливый", "Лживый"],
    ["Наивный", "Хитрый"],
    ["Общительный", "Застенчивый"]
]

result = []

# выбираем 3 разные пары
for pair_indices in itertools.combinations(range(len(pairs)), 3):
    selected_pairs = [pairs[i] for i in pair_indices]
    
    # выбираем по одному качеству из каждой пары
    for combo in itertools.product(*selected_pairs):
        result.append(list(combo))

# сохраняем в JSON
with open("traits_combinations.json", "w", encoding="utf-8") as f:
    json.dump(result, f, ensure_ascii=False, indent=2)

print(f"Сгенерировано комбинаций: {len(result)}")

И использует эти знания.
