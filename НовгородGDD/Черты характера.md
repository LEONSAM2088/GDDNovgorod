Игрок растёт **не потому, что набил уровень**, а потому, что **научился понимать людей**.

Он замечает  [[#Трейты]] NPC:

| Трейт            | Противоположный трейт  |
| ---------------- | ---------------------- |
| [[Амбициозен]]   | [[Малопритязательный]] |
| [[Щедрый]]       | [[Жадный]]             |
| [[Смелый]]       | [[Трусливый]]          |
| [[Миролюбивый]]  | [[Гневный]]            |
| [[Справедливый]] | [[Лживый]]             |
| [[Наивный]]      | [[Хитрый]]             |
| [[Общительный]]  | [[Застеничивый]]       |
Каждый NPC имеет комбинацию из 3 вариантов трейтов, однако не может иметь в комбинации трейты противоречащие друг другу, т.е нельзя быть амбициозным и малопритязательным одновременно.

Пример комбинаций:
Амбициозен + Щедрый + Смелый  
Амбициозен + Щедрый + Трусливый  
Амбициозен + Жадный + Смелый  
Амбициозен + Жадный + Трусливый

Амбициозен + Миролюбивый + Справедливый  
Амбициозен + Миролюбивый + Лживый  
Амбициозен + Гневный + Справедливый  
Амбициозен + Гневный + Лживый

Амбициозен + Наивный + Общительный  
Амбициозен + Наивный + Застенчивый  
Амбициозен + Хитрый + Общительный  
Амбициозен + Хитрый + Застенчивый

Малопритязательный + Щедрый + Смелый  
Малопритязательный + Щедрый + Трусливый  
Малопритязательный + Жадный + Смелый  
Малопритязательный + Жадный + Трусливый

Малопритязательный + Миролюбивый + Справедливый  
Малопритязательный + Миролюбивый + Лживый  
Малопритязательный + Гневный + Справедливый  
Малопритязательный + Гневный + Лживый

Амбициозен + Хитрый + Общительный → быстрый рост, агрессивная экспансия, риск скандалов.

Малопритязательный + Справедливый + Миролюбивый → стабильный “надёжный” NPC, любимец района.

Жадный + Хитрый + Общительный  
Агрессивный торговец: быстро реагирует на рынок, манипулирует ценами, может устраивать искусственный дефицит.

Трусливый + Наивный + Малопритязательный  
Пассивный NPC: почти не развивается, легко теряет ресурсы, но редко попадает в серьёзные конфликты.


import itertools
import json

pairs = [
    ["Амбициозен", "Малопритязательный"],
    ["Щедрый", "Жадный"],
    ["Смелый", "Трусливый"],
    ["Миролюбивый", "Гневный"],
    ["Справедливый", "Лживый"],
    ["Наивный", "Хитрый"],
    ["Общительный", "Застенчивый"]
]

result = []

# выбираем 3 разные пары
for pair_indices in itertools.combinations(range(len(pairs)), 3):
    selected_pairs = [pairs[i] for i in pair_indices]
    
    # выбираем по одному качеству из каждой пары
    for combo in itertools.product(*selected_pairs):
        result.append(list(combo))

# сохраняем в JSON
with open("traits_combinations.json", "w", encoding="utf-8") as f:
    json.dump(result, f, ensure_ascii=False, indent=2)

print(f"Сгенерировано комбинаций: {len(result)}")
