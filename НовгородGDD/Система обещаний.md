#Система

# Зачем:  

Для того что бы можно было заключать устные или узаконенные договора с персонажами, имея возможность выполнить обещание потом до окончания срока или рассчитывать на выполнения обещания других персонажей в будущем.

# Обещание игроку:


Я могу пообещать и/или взять обещание с другого персонажа для достижения поставленных целей. 

# Состояния (State)

# Правила

1. Если доверие персонажа A к персонажу B достаточно высокое, тогда создаётся контракт на выбранных условиях.
2. Если доверие околонулдевое, к значению доверия прибавляется оценка репутации. см. [[Система фактов о персонаже (Репутация)]]
3. Когда составляется договор, т.е. каждая сторона добавляет обязательства, у каждого обязательства есть вес, если вес обязательств A выше доверия B к A, тогда договор не может быть заключен.
4. Если тип договора  узаконенный (не устный), то добавляется бонус к доверию к обеим сторонам, либо к одной, если одна из сторон Игрок.
5. Если обязательства имеют незаконный характер, то договор не может быть узаконен.
6. Если сторона A выполняет условия контракта, тогда доверие персонажа B увеличивается на значение равное:
7. ```C#
   // вес обязательств = W
   // Кредит доверия = C
   var result = W * (1-(20/C))
   
   ```
8. Если сторона A не выполняет условия контракта, тогда доверие персонажа B уменьшается на значение равное:
9. ```C#
   // вес обязательств = W
   // Кредит доверия = C
   var result = W * 2
   
   ```
